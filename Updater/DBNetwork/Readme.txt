# DBNUpdaterBuildCaches

DBNUpdaterBuildCaches is an easy to use utility that builds cache information for the aIW3 updater.

## Usage

To use this utility, please follow these steps:

1. Build DBNUpdaterBuildCaches and any other required libraries.
2. Create a folder names caches and inside that create a sub-directory for your package (e.g. iw4-client).
3. Inside your package folder place all of the files that you wish to have compressed and packaged for update.
4. Then, run BuildCaches.exe and let it propagate.
5. Finally, copy all of the files and folders from CacheUpload to your web server.

## Other notes

Ensure that the version numbers are always consistent.
